1. **Push to GitHub**:
   ```bash
   git add .
   git commit -m "Ready for deploy"
   git push origin main
   ```

2. **Connect to Vercel**:
   - Go to [Vercel.com](https://vercel.com).
   - Click **"Add New"** > **"Project"**.
   - Import your `yumecav` repository.
   - Vercel will auto-detect **Vite**. Click **Deploy**.
